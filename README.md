
# 🧠 Gemini 2.0 E-commerce Product Listing Generator

This project showcases how to use Google’s Vertex AI SDK to interact with the Gemini 2.0 multimodal model for generating e-commerce product listings. By combining image inputs and product metadata, the model produces optimized titles, descriptions, and key attributes for online catalogs.

---

## 🔍 What is Gemini 2.0?

Gemini 2.0 is Google’s advanced multimodal large language model capable of processing and reasoning across text, images, audio, video, and code. With a context window of up to 1 million tokens, Gemini is ideal for complex, large-scale tasks such as product generation, analysis, and creative content development.

[Learn more about Gemini 2.0 →](https://cloud.google.com/vertex-ai/docs/generative-ai/learn/models/gemini)

---

## 📦 Project Objectives

* Upload an image of a product (e.g., apparel, electronics)
* Provide metadata like category, color, material, and type
* Use Gemini 2.0 to automatically generate:

  * A compelling product title
  * A detailed and marketable product description
  * Key attributes for listing (e.g., fabric, fit, use-case)

---

## 🛠️ Technologies Used

* Google Vertex AI SDK
* Gemini 2.0 Multimodal Model
* Google Cloud Platform (GCP)
* Python (for backend integration)
* PIL / Image handling libraries (optional)

---

## 📸 Workflow Summary

1. An image of the product is uploaded
2. Minimal metadata is supplied (e.g., “Blue cotton shirt”)
3. Gemini processes the image and metadata
4. Output includes:

   * 🏷️ Product Title
   * 📝 Product Description
   * 📋 Key Attributes (Color, Material, Fit, etc.)

---

## ✅ Sample Output

Input:

* Image: Blue Cotton Shirt
* Metadata: Category - Shirt, Color - Blue, Material - Cotton

Gemini 2.0 Output:

* Title: Classic Blue Cotton Casual Shirt
* Description: Crafted from soft, breathable cotton, this timeless blue shirt adds effortless style to any casual outfit.
* Attributes: Color: Blue, Material: Cotton, Fit: Regular, Occasion: Casual
